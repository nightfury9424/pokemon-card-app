## 데이터 무결성 원칙 (정책)

1. **scrydex chart = 1차 가격 출처** (자동 일별 수집)
2. **eBay sold listings = 무결성 검증** (변동 큰 경우 + 사용자에게 실거래 카운트 노출)
3. **실거래 0건 카드 = "추정가" 라벨 명시** — Raw NM 거래 30일 0건이면 가격에 노란 경고
4. **KO_EST forward-fill 금지** — 데이터 없는 날은 빈 채로. 사용자 화면은 가장 최근 KO_EST + "X일 전 기준" 표시
5. **KO_EST 산식**: `scrydex_raw × 환율 × 레어도별 계수` (단 이상치는 eBay 검증 통과한 것만 사용)
6. **scrydex publish 지연**은 정상 — 카드별 비동기 publish. `--days 3` sync로 며칠 안에 자동 보충
7. **차트 폭등/폭락 자동 cross-check** — 일별 변동 >30% 시 eBay sold listings로 검증, 미검증 시 KO_EST 제외
